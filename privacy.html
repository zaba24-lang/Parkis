Integritetspolicy — Parkis

Senast uppdaterad: 2026-02-13

1) Översikt

Parkis hjälper användare att tolka parkeringsskyltar genom bildanalys. När du använder appens analysfunktion behandlas de bilder du väljer att skicka in.

AI i Parkis: Huvudanalysen utförs av en intern AI-modell som är utvecklad och tränad av Parkis (skapad internt). Modellen är avsedd att känna igen och filtrera endast parkeringsskyltar och används inte för att identifiera personer.

Viktigt om tredjepart (AI): För att läsa text i bilder (OCR) kan vi, efter ditt samtycke, använda Google Cloud Vision. För att översätta extraherad text kan vi, efter ditt samtycke, använda DeepL. Analysresultat och appdata kan lagras i Google Cloud Firestore. Vi skickar inte data till dessa tjänster utan att du först ger ditt samtycke i appen.

2) Vilka uppgifter vi behandlar

  • Bilder (användarinnehåll): När du tar eller väljer en bild kan den laddas upp till vår server för att generera ett analysresultat som returneras till appen. Bilder kan innehålla personuppgifter om sådant råkar finnas i bilden (t.ex. registreringsskyltar).
  • Text i bilden (om relevant): Text kan extraheras som en del av analysen (t.ex. tider och villkor) och kan vid behov översättas.
  • Enhets-/app-identifierare (UUID): Parkis använder en slumpmässig app-identitet (UUID) lokalt på din enhet för att hantera credits, köpstatus och säkerhetskontroller. Ingen inloggning krävs.
  • Diagnostik: Tekniska loggar och felhändelser kan behandlas för att förbättra stabilitet och säkerhet.
  • Köp och credits: Uppgifter kopplade till köp (t.ex. abonnemangsstatus och nödvändiga transaktionsindikatorer) samt ditt credits-saldo behandlas för att leverera betalfunktioner och förhindra manipulation.

3) Hur vi använder uppgifterna

  • Appfunktionalitet: Bearbeta bilder på servern och leverera analysresultat.
  • Textförståelse (OCR) och översättning: Extrahera text från bilden och vid behov översätta text för att förbättra resultatets precision.
  • Valfri förbättring av AI (opt-in): Endast om du aktivt väljer detta i appen kan vissa bilder sparas temporärt för tester/experiment i syfte att förbättra vår interna AI-modell och tjänstens kvalitet.
  • Säkerhet och anti-fusk: Upptäcka och förhindra missbruk, automatiserade attacker och manipulation av credits/betalflöden.
  • Stabilitet: Felsöka, åtgärda buggar och förbättra prestanda.

3.1) Samtycke innan data delas med tredjeparts-AI

Innan vi skickar en bild eller text till en tredjeparts-AI-tjänst (t.ex. Google Cloud Vision för OCR eller DeepL för översättning) visar appen en tydlig samtyckesruta. Du kan välja Godkänn eller Inte nu. Om du väljer "Inte nu" skickas ingen data till tredjeparts-AI och online-analysen genomförs inte.

Du kan när som helst ändra ditt val i appens inställningar (om funktionen finns) eller genom att kontakta oss.

4) Lagringstid

Standard: Bilder som används för att ge dig ett analysresultat behandlas och raderas normalt så snart analysen är klar.

Om du har valt "Valfri förbättring av AI (opt-in)": Bilder som du delar för tester/utveckling kan lagras i upp till 3 månader och raderas därefter. Övriga tekniska loggar och säkerhetsrelaterade uppgifter behålls endast så länge som krävs för ändamålet eller enligt tillämpliga lagkrav.

5) Delning med tredje part

Vi säljer inte personuppgifter. När det behövs för att leverera appens funktion kan vi dela vissa uppgifter med följande leverantörer. Vi delar endast den minsta mängd data som krävs (data-minimering).

  • Google Cloud Vision (Google LLC) — OCR
    Vad kan skickas: bild (eller relevanta delar av bilden) för att extrahera text.
    Syfte: läsa text på skyltar/tilläggstavlor (t.ex. tider och villkor).
  • DeepL (DeepL SE) — Översättning
    Vad kan skickas: extraherad text (OCR-text) och språkinställningar.
    Syfte: översätta text för att presentera resultatet.
  • Google Cloud Firestore (Google LLC) — Lagring
    Vad kan lagras: analysresultat (t.ex. extraherad/översatt text och tolkade regler), UUID, credits-saldo och nödvändig köpstatus.
    Syfte: leverera historik/credits och driftsäker funktion.
  • RevenueCat, Inc. — Köp/abonnemang (verifiering)
    Vad kan delas: nödvändiga köpstatus-indikatorer (t.ex. abonnemangsstatus) och en app-identifierare (UUID/appUserID) för att kunna verifiera köp och hantera credits.
    Syfte: verifiera köpstatus och hantera credits korrekt.
  • Apple (App Store) — Betalningsbehandling
    Vad behandlas: köp genomförs via App Store. Vi tar inte emot eller lagrar fullständiga kortuppgifter.
    Syfte: genomföra och administrera köp.

6) Spårning och annonser

Parkis visar inga annonser i nuläget. Parkis spårar inte dig över andra företags appar och webbplatser. Om detta ändras i framtiden uppdateras denna policy, och eventuella tillstånd/val som krävs enligt iOS kommer att hanteras enligt gällande regler.

7) Betalningar via RevenueCat (credits)

Betalningar och abonnemang hanteras via App Store samt RevenueCat för att verifiera köpstatus och hantera credits. Vi lagrar inte fullständiga kortuppgifter. Vi kan lagra nödvändiga transaktionsindikatorer, köpstatus och credits-saldo på serversidan för att säkerställa korrekt saldo och förhindra manipulation.

8) Säkerhet och anti-fusk

Vi använder rimliga säkerhetsåtgärder (t.ex. krypterad överföring, åtkomstkontroller och serverbaserade valideringar) för att skydda data och minska risken för intrång och fusk. Försök att manipulera credits, betalflöden eller säkerhetsmekanismer kan leda till avstängning, och vid misstanke om bedrägeri eller intrång kan ärendet hanteras enligt tillämplig lag.

9) Rättslig grund (GDPR)

Vi behandlar uppgifter för att kunna leverera tjänsten (avtal) och för att skydda systemet mot missbruk (berättigat intresse).

Tredjeparts-AI (OCR/översättning): När Google Cloud Vision eller DeepL används görs det efter ditt samtycke i appen.

Valfri förbättring av AI (opt-in): Om du väljer att dela bilder för tester/AI-förbättring behandlar vi dessa uppgifter baserat på samtycke. Du kan när som helst återkalla detta samtycke.

10) Dina rättigheter

Du kan begära information om behandling kopplad till din app-identitet (UUID), begära rättelse eller radering när det är möjligt, samt invända mot viss behandling. Du kan även återkalla samtycke för tredjeparts-AI och för "Valfri förbättring av AI (opt-in)" när som helst. Kontakta oss via e-post nedan.

11) Barn

Appen är inte avsedd för barn under 13 år. Om du är vårdnadshavare och tror att ett barn har skickat in data av misstag, kontakta oss.

12) Ändringar i policyn

Vi kan uppdatera denna policy vid behov, exempelvis vid ändringar i funktioner, säkerhetsåtgärder eller lagkrav. Den senaste versionen publiceras och gäller från och med datumet "Senast uppdaterad".
